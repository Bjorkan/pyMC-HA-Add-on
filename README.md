# pyMC Home Assistant Add-on Repository

This repository contains Home Assistant add-ons for running pyMC services.

## Add-ons

### [`pymc_repeater_dev`](./pymc_repeater_dev)

Tracks the upstream `pymcdev/pymc-repeater:dev` image.

### [`pymc_repeater_main`](./pymc_repeater_main)

Tracks the upstream `pymcdev/pymc-repeater:main` image when that channel is
published upstream.

### [`pymc_glass_dev`](./pymc_glass_dev)

Builds and tracks the upstream
[`pyMC-Glass`](https://github.com/pyMC-dev/pyMC-Glass) `dev` branch.

### [`pymc_glass_main`](./pymc_glass_main)

Builds and tracks the upstream
[`pyMC-Glass`](https://github.com/pyMC-dev/pyMC-Glass) `main` branch.
